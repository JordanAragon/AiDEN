AiDEN 🌿

Sistema avanzado de gestión para viveros potenciado por Inteligencia Artificial.

# 🚀 Propósito
AiDEN nace como una respuesta tecnológica para la modernización de los invernaderos, permitiendo:
* **Monitoreo en tiempo real:** Control de estados y salud de las plantas.
* **Gestión Inteligente:** Automatización de registros y análisis de datos ambientales.
* **Diseño Semántico:** Interfaz profesional construida bajo estándares modernos para una experiencia de usuario fluida y accesible.

# 🛠️ Tecnologías destacadas
* IA para análisis predictivo.
* Maquetación con CSS Grid y HTML5 semántico (Zero-Div Policy).
* Enfoque en diseño minimalista y alto rendimiento.
